OVERVIEW

The MITRE ATT&CK (Adversarial Tactics, Techniques, and Common Knowledge)
framework is a globally-accessible knowledge base of adversary tactics and
techniques based on real-world observations. It is used as a foundation for
threat models and methodologies in the private sector, government, and the
cybersecurity product and service community.

ATT&CK Enterprise Matrix covers:
  - Windows, macOS, Linux environments
  - Cloud (AWS, Azure, GCP, SaaS, IaaS)
  - Network infrastructure
  - Containers

Current statistics (v14.1):
  - Tactics: 14
  - Techniques: 201
  - Sub-techniques: 424
  - Mitigations: 43
  - Groups: 138
  - Software: 740+

TA0043 — RECONNAISSANCE
  Definition: Gathering information to plan future adversary operations.
  Goal: Understand target's infrastructure, personnel, and security posture.
  Key techniques:
    T1595 — Active Scanning (port scanning, vulnerability scanning)
    T1592 — Gather Victim Host Information (hardware, software, firmware)
    T1589 — Gather Victim Identity Information (email, credentials, names)
    T1590 — Gather Victim Network Information (IP ranges, DNS, topology)
    T1591 — Gather Victim Org Information (business relationships, employees)
    T1598 — Phishing for Information (spearphishing for credentials/info)
    T1597 — Search Closed Sources (dark web, threat intel purchases)
    T1596 — Search Open Technical Databases (WHOIS, DNS, certificate CT logs)
    T1593 — Search Open Websites/Domains (social media, job postings)

TA0042 — RESOURCE DEVELOPMENT
  Definition: Establishing resources to support operations.
  Goal: Acquire infrastructure and capabilities before attacking.
  Key techniques:
    T1583 — Acquire Infrastructure (domains, VPS, botnets)
    T1586 — Compromise Accounts (social media, email)
    T1584 — Compromise Infrastructure (hijack legitimate servers)
    T1587 — Develop Capabilities (malware, exploits, certificates)
    T1585 — Establish Accounts (create fake personas)
    T1588 — Obtain Capabilities (buy malware, tools, exploits)

TA0001 — INITIAL ACCESS
  Definition: Techniques used to gain initial foothold in a network.
  Goal: Get inside the target environment.
  Key techniques:
    T1189 — Drive-by Compromise (malicious web content)
    T1190 — Exploit Public-Facing Application (web apps, VPNs, firewalls)
    T1133 — External Remote Services (VPN, RDP, Citrix)
    T1200 — Hardware Additions (malicious USB, network implant)
    T1566 — Phishing (spearphishing attachment, link, service)
    T1091 — Replication Through Removable Media
    T1195 — Supply Chain Compromise (software, hardware, dependencies)
    T1078 — Valid Accounts (default, domain, local, cloud)

TA0002 — EXECUTION
  Definition: Running adversary-controlled code on a local or remote system.
  Goal: Execute malicious code to accomplish objectives.
  Key techniques:
    T1059 — Command and Scripting Interpreter (PowerShell, Bash, Python, VBA)
    T1609 — Container Administration Command
    T1203 — Exploitation for Client Execution
    T1559 — Inter-Process Communication (DDE, XPC)
    T1106 — Native API (WinAPI, POSIX)
    T1053 — Scheduled Task/Job (cron, at, systemd timers)
    T1129 — Shared Modules (DLL loading)
    T1072 — Software Deployment Tools (SCCM, Ansible, Chef)
    T1569 — System Services (service execution, launchctl)
    T1204 — User Execution (malicious file/link)
    T1047 — Windows Management Instrumentation

  Goal: Ensure continued access to compromised systems.
  Key techniques:

    T1197 — BITS Jobs (Windows Background Intelligent Transfer)
    T1547 — Boot or Logon Autostart Execution (registry keys, startup folder)
    T1037 — Boot or Logon Initialization Scripts (logon scripts, rc.d)
    T1176 — Browser Extensions (malicious extensions)
    T1554 — Compromise Host Software Binary (trojanize legitimate binary)
    T1136 — Create Account (local, domain, cloud)
    T1543 — Create or Modify System Process (services, launchd, systemd)
    T1546 — Event Triggered Execution (COM hijacking, .bash_profile)
    T1133 — External Remote Services
    T1574 — Hijack Execution Flow (DLL hijacking, PATH manipulation)
    T1525 — Implant Internal Image (container/VM image)
    T1556 — Modify Authentication Process (password filter DLL, MFA bypass)
    T1137 — Office Application Startup (templates, add-ins)
    T1542 — Pre-OS Boot (bootkit, UEFI implant)
    T1053 — Scheduled Task/Job
    T1505 — Server Software Component (web shell, IIS module, SQL procedure)
    T1078 — Valid Accounts

    T1053 — Scheduled Task/Job
    T1078 — Valid Accounts
